# Case Templates Chunk Notes

These notes break the workflow into safe, small checkpoints.

## Chunk 1: Login + Navigation
1. Login with valid user.
2. Open Dashboard.
3. Click Templates in the left sidebar.
4. Confirm URL is exactly `https://app.thecasework.com/templates`.
5. If not already there, click `Case Templates` tab.

## Chunk 2: Copy Rows
1. Copy `Pikachu` and confirm `Pikachu-Copy` exists.
2. Copy `You're the Best Around` and confirm `You're the Best Around-Copy` exists.
3. Copy `You're the Best Around-Copy` and confirm `You're the Best Around-Copy-Copy` exists.

## Chunk 3: Delete Only Created Copies
1. Delete `Pikachu-Copy`.
2. Delete the extra `You're the Best Around-Copy` created in this run.
3. Delete `You're the Best Around-Copy-Copy`.
4. Keep final baseline rows:
   - `Pikachu`
   - `You're the Best Around`
   - `You're the Best Around-Copy`

## Chunk 4: Full Edit Flow (Per Row)
Run full Add/Edit flow for each row:
1. `Pikachu`
2. `You're the Best Around`
3. `You're the Best Around-Copy`

Flow includes:
- Basic fields
- Info icon checks
- Allowed Statuses (New, Active, Completed, Closed, Removed)
- Description and Overview fields
- Initial Note
- Engagement Template options
- Milestone + Event + Task popups
- Save
- Re-open Edit and click Back to Case Templates

## Chunk 5: New Template Full Form + Back
1. Click `New Template`.
2. Fill all same fields and popup workflows.
3. Click `Back to Case Templates` (not Save) at the end.
